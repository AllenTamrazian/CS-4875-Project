Module 11 - Lambda Expressions

Introduction

So, let's say that you want to do a simple operation on a list. You could use a for loop, covered in the Functions lesson, or we could do it in a much simpler way using something called lambda expressions. Lambda expressions are basically like functions that don't have a name and don't look like typical functions with the block format. Lambda expressions also can be treated as a normal object, which lets use do things such as pass them as input parameters, return them, and much more.
